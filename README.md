# 🔧 AI Вибродиагностика PRO

**Интерактивный учебный стенд по методу спектра огибающей для диагностики подшипников качения.**

Основано на полном практическом руководстве для виброанализатора СД-23 (ВАСТ) и ПО DREAM32.

---

## 🎯 Для кого это приложение?

- Инженеры-вибродиагносты
- Студенты технических вузов
- Специалисты КДЦ (контрольно-диагностических центров)
- Все, кто хочет изучить метод спектра огибающей

---

## 🚀 Возможности приложения

### 📊 Генератор спектров огибающей
- 11 типов подшипников (6200, 6300, роликовые, высокоскоростные)
- 4 типа дефектов: BPFO, BPFI, BSF, FTF
- 4 стадии развития дефекта
- Выбор фильтра (1/3 октавы, 1 октава)
- Выбор центральной частоты fc (800-6300 Гц)
- 4 графика: сигнал, огибающая, линейный и логарифмический спектр
- AI Диагност с рекомендациями

### 📖 Книга (12 разделов + 4 приложения)
Полное руководство (~300 страниц):
1. Теоретические основы метода огибающей
2. Аппаратное обеспечение (СД-23, датчики)
3. Настройка параметров измерения
4. Расчёт дефектных частот (BPFO, BPFI, BSF, FTF)
5. Интерпретация спектров
6. 20 практических кейсов
7. Специфические случаи (тихоходные, ЧРП, кавитация)
8. Трендинг и прогнозирование
9. Стандарты ISO/ГОСТ
10. Программное обеспечение (DREAM32)
11. 7 типичных ошибок аналитиков
12. Инструменты и ресурсы

**Приложения:**
- А. Таблицы частот для подшипников
- Б. Шаблоны отчётов
- В. Чек-листы для полевого инженера
- Д. Словарь терминов (русско-английский)

### 🤖 AI-агенты
- **AI Диагност** — анализ спектра с рекомендациями
- **AI Наставник** — персональные советы по уровню подготовки
- **AI Экзаменатор** — адаптивные вопросы

### 📝 AI Экзамен
- Адаптивная сложность (Новичок → Эксперт)
- Отслеживание прогресса
- План самообразования на 4 года

### 📋 Чек-листы
Интерактивные чек-листы для:
- Подготовки к выходу на объект
- Проведения измерения
- Анализа данных в офисе
- Самопроверки типичных ошибок

### 🔍 Поиск по книге
Полнотекстовый поиск по всем 12 разделам

## 🎯 Как использовать

### Для начинающих:
1. Начните с раздела **"📖 Книга"** → "1. Теоретические основы"
2. Изучите раздел "4. Расчёт частот"
3. Перейдите в **"📊 Генератор спектров"** — поиграйте с параметрами
4. Перейдите в **"📝 AI Экзамен"** — проверьте знания

### Для опытных:
1. Используйте генератор для моделирования реальных случаев
2. Изучите чек-листы для полевой работы
3. Используйте шаблоны отчётов для оформления результатов
4. Проверяйте себя через поиск по книге
